# WhatsApp Automation Project 🚀

## 📌 About
This project sends automated WhatsApp messages using Python.

## ⚙️ Features
- Send WhatsApp messages automatically
- Schedule messages
- Simple input-based system

## 🛠️ Tech Stack
- Python
- PyWhatKit
- PyAutoGUI

## 📦 Installation
```bash
pip install pywhatkit pyautogui
```

## ▶️ How to Run
```bash
python Whatapp.py
```
